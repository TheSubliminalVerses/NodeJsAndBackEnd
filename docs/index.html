API Documentation

This API is a standards RESTful API that allows you to view,create,update and delete products. Below you will find documentation on all objects and methods of the API.

The API's base url is http://localhost:3000/apiv1

No authentication or HTTP headers are required to access the API. All body input should be given in application/json format and all data is returned in application/json format.

Generic

Response Codes
Various response codes and their meanings
code name description
200 OK Successful, with content
201 Created A resource was created
204 No Content Successful, no content
304 Not modified Content was not modified even though request was made to do so.
400 Bad request It was not possible to process request. Review required inputs
404 Not found Requested resource was not found

Objects

Product
The product object describes a product.

Attributes

key type description
id number The product's identifier
brand string The product's brand
model string The product's model
os string The product's operating system
screensize number The product's screensize in inches
image string Image URL of the product

Methods

POST /products
Create a single product

Body attributes

The possible input parameters to create a product
key type description
brand* string The product's brand
model* string The product's model
os* string The product's operating system
screensize number The product's screensize in inches
image string Image URL of the product
*required

Example Body

                
    {
      "brand": "Product 1",
      "model": "This is product 1",
      "os": "ios"
      "screensize": "10.00",
      "image": "http://placehold.it/200x200"
    }

Responses

400 - Bad request
Invalid input, review your input body
                  
  // Empty
200 - Success
Returns the newly created product.
                  
    {
      "id": 1,
      "brand": "Product 1",
      "model": "This is product 1",
      "os": "ios"
      "screensize": "10.00",
      "image": "http://placehold.it/200x200"
    }
